♣ Problem 9.5   (Ans: (a) RD = 10[KΩ])
(b) Cs = 3.536[uF]
(c) ωz = 10[Hz]
(d) fL = 100[Hz]
(e)Av(DC) = -2[V/V])

공통소스 증폭기에서 mid-band 의 AM = -gmRD 이며, AM = -20[V/V],
gm=2[mA/V] 이므로 RD = 10[KΩ] 이다.
ωL = 1/RC 이므로 fL = 1/(2πRC) 이다. 이때 R은 1/gm // RS 이며 100[Hz] 가 나오려면
Cs = 3.536[uF] 이다.
ωz = 1/CsRs = 10[Hz] 이다.
fL 은 ωP2 / 2π 와 거의 같으므로 100[Hz] 에 가깝다.
직류이득을 구하기 위해 커패시턴스를 개방시키면 1/gm 과 RS 가 직렬이 되고, 전압이득은 -gm(1/gm+RS) 이다.
따라서 대입시 -2[V/V] 의 전압이득을 가지게 된다. 기존 전압이득 -20 의 magnitude 를 20logAM 에 대입시 26[dB] 이 나오며
직류 전압이득을 대입시 6[dB] 이 나오게 된다.

♣ Problem 9.10   (Ans: (a)AM = -RC/re+Re, fL=1/2π(re+Re)CE)
(b)AM = (-RC / re)(1/(1+Re/re))
(c)fL = (1/2π(re+Re)CE)*(1/(1+Re/re))
(d)Re = 900[Ω] , AM = 10[V/V])

AM = -RC/re+Re 이며 fL=1/2π(re+Re)CE 이다.
Re 의 영향을 보기위해 특정 factor 를 만들면, (-RC / re)(1/(1+Re/re)) 로 묶어낼 수 있다.
fL 도 마찬가지로 (1/2π(re+Re)CE)*(1/(1+Re/re)) 이다.
factor 가 (1/(1+Re/re)) 임을 알았고, 1/10 배의 factor 을 구하면 Re = 900[Ω] 이다.
또한, AM = 10[V/V] 가 된다.

♣ Problem 9.19   (Ans: fT = 5.06[GHz])

gm = 40[mA/V], rπ = 2.5[KΩ], ro = 50[KΩ] 임을 25[mV], 주어진 IC 로 구하고,
Cde = τFgm = 1.2[pF] 이며 Cje = 2Cje0 = 40[fF] 이다
따라서 Cπ = Cje + Cde = 1.24[pF] 이 되며,
Cμ = Cμ0 / (1+VCB/V0c)m 대입시 15.667[fF] 이다.
따라서 fT = gm / 2π(Cπ + Cμ) 대입시 5.06[GHz] 이다.
