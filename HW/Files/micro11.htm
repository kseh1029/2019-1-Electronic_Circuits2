♣ Problem 10.50  

Vs = 0, Ro = Vx / Ix
Ve = Ix[Re || (rπ + Rb)]
Ib = - Ix * Re / (Re + rπ + Rb)
Vx = (Ix - βIb)ro + Ve
Ro = Vx / Ix = ro + [Re || (rπ + Rb)][1 + ro * β / (rπ + Rb)] = Vx / Ix = ro + [Re || (rπ + Rb)][1 + gmro * rπ / (rπ + Rb)]
Rb = 0, Ro = ro + (Re || rπ)(1 + gmro)
Re >> rπ 면 Ro 은 최대값이 된다.
Ro,max = ro + rπ(1 + gmro) ≅ βro
