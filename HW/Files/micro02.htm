♣ Problem 8.17   (Ans: (a) Ad= gm1,2*(1/gm3,4//ro3,4//ro1,2)
(b) root(un(W/L)1,2 / un(W/L)3,4)
(c) 25

차동회로의 공통 접지가 있으므로 회로를 절반으로 나누어서 공통소스 증폭기처럼 해석을 해보면
RD = 1/gm3 // ro3 이며 접지된 저항끼리 병렬하면
Ad= gm1,2*(1/gm3//ro3//ro1) 이 된다.
이제 두 회로가 정합되어있으므로 알맞게 표현을 하면
∴ Ad= gm1,2*(1/gm3,4//ro3,4//ro1,2)
이 식에서 ro를 무시하게 되면 Ad = gm1,2/gm3,4
gm = root(2unCox(W/L)(I/2)) 이므로 각각 gm 에 대입시
∴ Ad = root(un(W/L)1,2 / up(W/L)3,4)
여기서 un = 4up 이고 Ad = 10 이라면
∴ (W/L)1,2 / (W/L)3,4 = 25

♣ Problem 8.19   (Ans: (a) VS1 = VS2 = -(VOV1,2 + Vtn)
(b)VOV3 = VC + VOV1,2
(c)Rs = 1/gm1,2 * (VOV1,2/VOV3)
(d1) VC = 0
(d2) VOV3 = VOV1,2

VG1 = VG2 = 0 이므로 VGS1 = VGS2= VOV1,2 + Vtn
즉 VS1 = VS2 = -(VOV1,2 + Vtn) 이다.
이 상황에서 VDS 가 0라면, Q3 에는 전류가 안흐르고 과구동전압은 VC = VS1,2 - Vtn 이므로
∴ VC + VOV1,2
VG1 에 Vid/2 VG2에 -Vid/2 라 한다면 Q3는 Triode Region 에 있으므로 저항처럼 작동한다.
즉 Rs = 1 / unCox(W/L)3VOV3 이고
gm1,2 = unCox(W/L)1,2VOV1,2
gm3 = unCox(W/L)3VOV3 이 세 식에 (W/L)3 = (W/L)1,2대입시
∴ Rs = 1/gm1,2 * (VOV1,2/VOV3)
Rs = 1/gm1,2 일때 VOV3 = VOV1,2 이므로
∴ VC = 0
Rs = 0.5/gm1,2 일때 VOV3 = VC + 2VOV1,2 이므로
∴ VOV3 = VOV1,2
