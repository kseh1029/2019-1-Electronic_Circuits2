♣ Problem 2.20   (Ans: (a) Rm = -Rf, Ri = 0)
(b) Rm = -Rf/(1/A + 1), Ri = Rf/(A + 1)

(a) infinite 일때, (V2-V1)A=Vo 이므로 V2-V1)=0, 즉 V2=V1
이때 Vi=0 이므로 (Vi-Vo)/Rf = i1 = -Vo/Rf 이므로
Rm = -Rf, Ri = 0

(b) finite 일때, (V2-V1)A=Vo 에서 V2 = 0 이면 Vi = -Vo/A 이다.
(Vi-Vo)/Rf= i1=-Vo(1/A + 1) / Rf = iI 이므로
Rm = -Rf/(1/A + 1) 이며 Ri = Rf/(A + 1)

♣ Problem 2.32   (Ans: (a) R = 1[Kohm])
(b) input 0 ohm, output infinite ohm
(c) -0.571 mA ~ 0.571 mA
(d) 2.2mA

(a) iL / iI = 11[A/A] 이므로 -Vx/10k옴 * 10 = -Vx / R 이므로, R=1K옴

(b) input 은 가상접지로 인해 0옴이며, output 은 RL 이 독립적이므로 infinite 옴이다.

(c) -Vx/10Kohm + -Vx/1Kohm = (Vx - 12) / 1Kohm 일때 Vx = 40/7 이며

-Vx/10Kohm + -Vx/1Kohm = (Vx + 12) / 1Kohm 일때 Vx = -40/7 이므로
전류는 -0.571 mA ~ 0.571 mA 이다.
