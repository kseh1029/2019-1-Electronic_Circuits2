♣ Problem 2.65   (Ans: 756[Ω])

(V1 - βVo2 / 2) / R = (βVo1/2 - Vo) 이므로 Vo1 = v1 / (β-1)
Vo2 = V2 / (1-β) 이므로 Vo = Vo1 + Vo2 = v1/(β-1) + v2/(β-1)
Ad = 1/(1-β) = 10 이므로 β = 0.9 = R6/(R5+R6)
