♣ Problem 8.42   (Ans: Ad = αRc/re+Re
Rin=2(β+1)(re+Re)
(a) VCM-VBE-(I/2)Re
(b) VCM-VBE)

(a) 회로의 소신호 해석을 해보면 Vod=2αieRc 이다.
이때 ie = I/2 이므로 Vid=2(re+Re)ie 에 대입시
Vid=2(re+Re)(I/2) 를 얻을 수 있다.
Vod=αIeRc 에 대입하면 Vod= αVidRc/re+Re 가 나온다
즉 Ad = αRc/re+Re 이다.
Vid=2(re+Re)ib(β+1) 에서
Rin=2(β+1)(re+Re) 임을 알 수 있다.
DC 해석을 해보면 전류원 상단 전압은 VCM-VBE-(I/2)Re 이다.
(b) 회로의 소신호 해석을 하면 (a) 회로와 마찬가지로 나오게 되며, 입력저항또한 동일하다
다만, DC 해석시 전류원 상단 전압이 VCM-VBE 이다.

♣ Problem 8.51   (Ans: (a) VCM = 2.332[V]
(b) RD = 5.06[kΩ]
(c) VD = 2.47[V]
(d) ΔVD1 / ΔVCM = -1.92[V]
(e) ΔVCM = 0.287[V])

Id = (1/2)(kn’(W/L)(VOV2) 에 주어진 값 대입시
0.5 = (1/2)(2.5)(VOV2 에서 VOV = 0.632[V]
VOV = VGS - Vtn 이므로 0.632 = VGS - 0.7, VGS = 1.332[V]
VCM - VGS = VS 이므로 VCM - 1.332 = 1, VCM = 2.332[V]
Ad = gmRD = 8[V/V] 이며 gm = 2ID / VOV = I/VOV = 1[mA] / 0.632[V] = 1.58[mS]
따라서 RD = 5.06[kΩ]
VD = VDD - IDRD = 5 - 0.5 * 5.06 = 2.47[V]
ΔVD1 / ΔVCM = -(id)RD / (1/gm) + 2RSS
= -1.92[V]
VCM + ΔVCM = Vt + VD1 + ΔVD1 에 값 대입시
2.332 + ΔVCM= 0.7 + 2.47 + ΔVD1, ΔVD1 / ΔVCM = -1.92 이므로
ΔVCM = 0.287[V]

♣ Problem 8.74   (Ans: (a) x=0.3 [kΩ]
(b) x=0.225 [kΩ])

RC1 = 1.04 * 5 = 5.2 [kΩ], RC2 = 0.96 * 5 = 4.8 [kΩ] 이므로
RC1 + x = RC2 + (1-x) 에 대입시 x=0.225 [kΩ]
IS1 = 1.05IS, IS2 = 0.95IS 이므로
IE1 = 0.5 * 1.05 = 0.525, IE2 = 0.5 * 0.95 = 0.475
Vod = 0 이기 위해 VC1 = VC2 이여야 하므로
VCC - (RC1+x)IC1 = VCC - (RC2 + 1 - x)IC2
대입시 0.525(5+x) = 0.475(1-x), x=0.225 [kΩ]
